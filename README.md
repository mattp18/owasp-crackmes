# OWASP CrackMe Writeups

This repository contains structured reverse engineering writeups for OWASP Android CrackMe challenges.  
Each challenge includes analysis, exploitation approach, and supporting notes/screenshots.

---

## 🧠 Focus Areas

- Android reverse engineering (jadx, apktool)
- Static & dynamic analysis
- Smali-level patching and understanding
- Runtime instrumentation (Frida where applicable)
- Security control bypass techniques

---

## 📁 Challenges

| Challenge | Description | Key Techniques |
|-----------|-------------|----------------|
| Level 1   | Input validation bypass | Static analysis, String comparison, Binary patching |

> Each folder contains a full breakdown with screenshots, explanations, and any scripts used.
